# 🌦️ Weather Dashboard

Weather Dashboard is a modern web application that allows users to view real-time weather information for cities around the world. It provides essential weather details through a clean and responsive interface, making it easy to stay updated with current weather conditions.

---

## ✨ Features

- 🌍 Search weather by city
- 🌡️ Real-time temperature display
- 💧 Humidity information
- 🌬️ Wind speed details
- ☁️ Weather condition with icons
- 📱 Fully responsive design
- ⚡ Fast and lightweight interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Weather API (OpenWeatherMap or equivalent)

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project folder.

3. Open `index.html` using your browser or Live Server.

---

## ▶️ Usage

- Launch the application.
- Enter a city name.
- View current weather information instantly.
- Search different cities as needed.
