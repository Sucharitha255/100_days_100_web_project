A modern **Airbnb-inspired travel booking website** that allows users to explore destinations, browse beautiful property listings, search accommodations, and enjoy a clean, responsive user experience.

---

# ✨ Features

- 🏡 Modern Airbnb-inspired interface
- 🔍 Search bar
- 🌙 Dark / Light Theme Toggle
- 📱 Fully Responsive Layout
- 🖼 Beautiful Property Cards
- 🏖 Multiple Categories
- 🔐 Login Page
- 📝 Signup Page
- 📄 Property Details Page
- ⚡ Fast and Lightweight
- 🎨 Clean UI Design

# 🛠 Installation

Clone the repository

```bash
git clone https://github.com/your-username/WanderLust.git
```

Go into the project

```bash
cd WanderLust
```

Run with Live Server

OR

Open `index.html` directly in your browser.
