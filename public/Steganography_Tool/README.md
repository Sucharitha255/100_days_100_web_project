# 🔐 Steganography Tool

Steganography Tool is a web application that allows users to hide and extract secret messages inside images using steganography techniques. It demonstrates how information can be securely embedded into digital media without visibly altering the original image.

---

## ✨ Features

- 🔒 Hide secret text inside an image
- 🔓 Extract hidden messages
- 🖼️ Image-based steganography
- ⚡ Fast client-side processing
- 📱 Responsive user interface
- 🎨 Clean and intuitive design

## ▶️ Usage

1. Open the application.
2. Upload an image.
3. Enter the secret message.
4. Encode the message into the image.
5. Download or save the encoded image.
6. Upload an encoded image to decode and reveal the hidden message.

---

## 🔐 How Steganography Works

Steganography hides information within digital files by modifying image data in a way that is generally imperceptible to the human eye, allowing secure transmission of hidden messages.
