# Flashcard Quiz App

A modern flashcard quiz application built with HTML, CSS and Vanilla JavaScript.

## Features

- Multiple subjects:
  - Web Development
  - DBMS
  - Computer Networks
  - Operating Systems
  - DSA
- Interactive flashcards with Previous/Next navigation
- Hint system with custom or auto-generated hints
- Timed Challenge Mode (30, 60 and 120 seconds)
- Live statistics (Correct, Wrong, Attempted, Accuracy, Best Score)
- Results modal after each challenge
- Add, edit and delete flashcards
- Subject-wise flashcard support
- Local Storage persistence
- Responsive light-themed UI

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript (ES6)
- Local Storage API
- Google Fonts (Poppins)

## How to Use

1. Select a subject from the navigation bar.
2. Read the current flashcard.
3. Type your answer.
4. Use **Hint** if needed.
5. Check your answer.
6. Navigate using Previous/Next.
7. Manage flashcards using Add, Edit and Delete.
8. Start Timed Challenge to test your speed.
