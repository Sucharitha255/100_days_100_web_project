# ⚖️ VitePulse BMI Calculator

VitePulse BMI Calculator is a modern web application that helps users calculate their **Body Mass Index (BMI)** using height and weight inputs. It provides instant results along with BMI category information to promote better health awareness.

---

## ✨ Features

- ⚖️ Accurate BMI calculation
- 📊 Instant BMI result
- 🩺 BMI category classification
- 📱 Responsive user interface
- ⚡ Fast performance with Vite
- 🎨 Clean and modern design

---

## 🛠️ Technologies Used

- Vite
- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd VitePulse-bmicalculator
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

---

## ▶️ Usage

1. Launch the application.
2. Enter your height and weight.
3. Click the calculate button.
4. View your BMI value and health category instantly.

---

## 📖 BMI Categories

| BMI Range | Category |
|-----------|----------|
| Below 18.5 | Underweight |
| 18.5 – 24.9 | Normal Weight |
| 25 – 29.9 | Overweight |
| 30 and above | Obesity |
