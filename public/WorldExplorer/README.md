# Interactive World Explorer

A beautiful, modern web application to explore countries around the world with search, filtering, sorting, favourites, and country comparison!

## Features

- 🚀 **Search Countries**: Find countries by name quickly
- 🌍 **Filter by Region**: Filter countries by continent
- 📊 **Sorting**: Sort by name (asc/desc) or population (most/least)
- ⭐ **Favourites**: Save your favourite countries for quick access
- 🆚 **Country Comparison**: Compare up to 2 countries side by side
- 📱 **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- 🌓 **Dark/Light Mode**: Toggle between dark and light themes
- 💾 **LocalStorage**: Your favourites and comparisons are saved automatically
- 📄 **Country Details**: View detailed info about each country in a modal
- 🎨 **Glassmorphism UI**: Modern, sleek design with glass effects



## How to Use

1. Open `index.html` in your browser, or run a local server
2. Browse countries or use the search bar to find a specific one
3. Filter by region or sort to organize the countries
4. Click the star icon on any card to add it to favourites
5. Click "Compare" on any card to select it for comparison
6. View the comparison panel at the bottom of the page
7. Click "Clear" to reset the comparison

## Features in Detail

### Search
- Type in the search bar to find countries by name
- Results update in real-time

### Filtering
- Choose from: All, Asia, Europe, Africa, Americas, Oceania, Antarctic

### Sorting
- A to Z
- Z to A
- Most populated
- Least populated

### Comparison
- Select up to 2 countries to compare
- View side-by-side cards with:
  - Flag
  - Country name
  - Capital
  - Population
  - Region
  - Subregion
  - Area
  - Languages
  - Currency
  - Timezones
