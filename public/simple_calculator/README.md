# 🧮 Simple Calculator

Simple Calculator is a lightweight web application that performs basic arithmetic operations through an intuitive and responsive user interface. It is designed as a beginner-friendly project to demonstrate fundamental JavaScript concepts and DOM manipulation.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear calculator display
- ⚡ Instant calculations
- 📱 Responsive design
- 🎨 Clean and user-friendly interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd simple_calculator
```

Run locally

- Open `index.html` in your browser
- OR use Live Server in Visual Studio Code

---

## ▶️ Usage

1. Open the calculator.
2. Click the number buttons.
3. Choose an arithmetic operator.
4. Press `=` to calculate the result.
5. Use `C` or `AC` to clear the display.
