# 🛣️ Roadmap Progress Tracker

Roadmap Progress Tracker is a web application that helps users track their learning journey by organizing topics, monitoring completion status, and visualizing overall progress. It is ideal for students, developers, and self-learners following structured learning roadmaps.

---

## ✨ Features

- 📋 Track roadmap progress
- ✅ Mark completed topics
- 📊 Visual progress indicators
- 🎯 Goal-oriented learning
- 📱 Responsive interface
- 🎨 Clean and modern UI

---

## 🛠️ Technologies Used

- CSS3

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd "Roadmap Progress Tracker"
```

Run locally

- Open `index.html` in your browser
- OR use Live Server in Visual Studio Code

## ▶️ Usage

1. Launch the application.
2. Browse the roadmap sections.
3. Mark completed topics.
4. Track your learning progress visually.
5. Continue updating your roadmap as you learn.
