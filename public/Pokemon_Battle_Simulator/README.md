# Pokemon Battle Simulator

A retro, GBA-style interactive Pokemon battle simulator built using HTML, CSS, and Vanilla JavaScript. 

## Features
- Fetches random Generation 1 Pokemon dynamically using the [PokeAPI](https://pokeapi.co/).
- Classic turn-based attack system with randomized damage calculations.
- Dynamic health bars that change color (Green -> Yellow -> Red) based on remaining HP.
- CSS animations for attack impact (shake effects).

## Tech Stack
- HTML5
- CSS
- JavaScript
