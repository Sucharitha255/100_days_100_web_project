# ✈️ FlyTravel – Travel Booking Website (Day‑XX)

## 🌍 Overview
**FlyTravel** is a responsive web application designed to simplify **flight and hotel bookings**.  
It provides users with an intuitive interface, authentication system, and chatbot assistance, making travel planning seamless and engaging.

---

## ✨ Features
✅ Search and book flights, hotels, and more  
✅ User authentication (login/signup)  
✅ Integrated chatbot for travel assistance  
✅ Responsive design for all devices  
✅ Clean and modular project structure  
✅ Beginner‑friendly implementation with maintainable code  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Node.js (server-side logic)  
- Express.js  
- Environment variables for configuration  
- JSON for package management  

---

## ▶️ How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project.git
```
2. Navigate to project folder:
```bash
cd 100_days_100_web_project/public/Travel_booking_website
```
3. Install dependencies:
```bash
npm install
```
4. Run the server:
```bash
node server.js
```
5. Open in browser:
   - Visit `http://127.0.0.1:3000` (or configured port)
