# 📖 Story

Story is a simple and interactive web application designed to present engaging stories in a clean and user-friendly interface. It demonstrates front-end development concepts while providing an enjoyable reading experience.

---

## ✨ Features

- 📚 Interactive story interface
- 🎨 Clean and responsive design
- ⚡ Lightweight and fast
- 📱 Mobile-friendly layout
- 💻 Easy to customize

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 📂 Project Structure

```text
Story/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

---

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project folder

```bash
cd Story
```

Run locally

- Open `index.html` in your browser
- OR use Live Server in Visual Studio Code

---

## ▶️ Usage

1. Open the application.
2. Read and interact with the story.
3. Explore the available content.
