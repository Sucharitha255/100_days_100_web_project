# 🌿 Overview

**ZenSpace – Focus Dashboard** is a distraction-free productivity web application that combines a **Pomodoro timer**, **ambient sound mixer**, **Deep Focus Mode**, and **quick note-taking** into a single elegant workspace.

Designed with a clean, calming interface, ZenSpace helps students, developers, writers, and professionals maintain concentration while creating an immersive work environment.

Everything runs entirely in the browser with no backend or installation required.

---

# ✨ Features

## 🍅 Pomodoro Focus Timer

A beautiful circular countdown timer designed for productive work sessions.

Features include:

* 25-minute Focus session
* 5-minute Break session
* Custom session duration
* Start, Pause, and Reset controls
* Animated circular progress ring
* Automatic countdown updates
* Session completion notification

---

## 🎯 Strict Deep Focus Mode

A unique productivity feature that minimizes distractions.

When enabled:

* Session duration cannot be changed
* Pause button is disabled
* Reset becomes **"☠ Give Up"**
* Users must type **"quit"** to exit the focus session
* Prevents accidental interruptions

Perfect for deep work and uninterrupted study sessions.

---

## 🎵 Ambient Sound Mixer

Create your ideal working atmosphere by mixing relaxing background sounds.

Available soundscapes:

* 🌧️ Rain
* ☕ Café
* 🌊 Ocean Waves

Features:

* Independent volume sliders
* Simultaneous audio playback
* Looping ambient sounds
* Instant volume adjustment

---

## 🌗 Theme Toggle

Switch between beautiful visual themes.

Themes include:

* 🌙 Cozy Theme
* ☀️ Café Theme

The interface updates instantly for a fresh working environment.

---

## 📝 Smart Quick Notes

Capture ideas without leaving your workspace.

Features include:

* Auto-saving notes
* Character counter
* Save status indicator
* Clear notes option
* Local Storage support

Your notes remain available even after refreshing the page.

---

## 🔔 Session Completion Toast

When a focus session finishes:

* A clean toast notification appears
* Encourages taking a healthy break
* Automatically disappears after a few seconds

---

## 💾 Local Storage Support

ZenSpace automatically saves your notes in the browser using **Local Storage**.

No login or internet connection is required.

# 🎯 How to Use

1. Choose a focus or break session.
2. Enable **Strict Deep Focus Mode** if desired.
3. Click **Start** to begin the timer.
4. Adjust ambient sound volumes to create your ideal workspace.
5. Switch between Cozy and Café themes.
6. Write quick notes during your session.
7. Receive a notification when your session ends.
