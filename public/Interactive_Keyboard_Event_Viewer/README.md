# Interactive JavaScript KeyCode & Keyboard Event Viewer

A fast, visually pleasing, full-screen interactive utility that displays exact keyboard event data dynamically when a user presses any key.

## 🌟 Features
- **Dynamic Display:** Press any key to instantly view its large numeric `keyCode`.
- **Event Properties:** Renders exact data for `event.key`, `event.location`, `event.code`, and `event.which`.
- **Default Action Prevention:** Intelligently prevents default browser actions for keys like spacebar, arrows, and slash so the tool doesn't scroll or trigger searches while testing.
- **Key History:** Maintains a real-time list of the last 5 keys pressed at the bottom of the screen.
- **Premium UI:** Uses a modern dark theme with glassmorphism, responsive DOM cards, and a pulsating initial state.
