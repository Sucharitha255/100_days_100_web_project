# 💻 Web Kernel Simulator

Web Kernel Simulator is a browser-based simulation that demonstrates how a simple operating system kernel works. It provides an interactive interface for understanding core operating system concepts such as process execution, command handling, memory simulation, and system behavior in a visual and beginner-friendly way.

---

## ✨ Features

- 🖥️ Interactive kernel simulation
- ⚡ Command execution interface
- 📂 Simulated operating system environment
- 🎨 Clean and responsive user interface
- 📚 Beginner-friendly educational project
- 🚀 Lightweight implementation using web technologies

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory.

3. Open `index.html` in your browser or run it using Live Server.

## ▶️ Usage

- Launch the simulator.
