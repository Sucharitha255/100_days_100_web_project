
# Particle Wave Animation

## 📖 Description
Particle Wave Animation is a creative frontend project that renders a full‑screen canvas filled with colorful particles flowing in wave‑like motion. The particles move fluidly using sine and cosine functions, wrap around screen edges, and continuously shift hues to create a mesmerizing rainbow effect.

## ✨ Features
- Full‑screen animated particle canvas
- 200 particles with random positions and sizes
- Wave‑like motion driven by sine and cosine functions
- Dynamic rainbow coloring using HSL hues
- Smooth looping animation with `requestAnimationFrame`
- Responsive resizing to fit any browser window
- Pure **HTML**, **CSS**, and **JavaScript (ES6)** implementation

## ⚙️ Installation / Setup
1. Clone or download this project.  
2. Open the project folder.  
3. Ensure the following files are in the same directory:
   - `index.html`
   - `style.css` (optional if separated)
   - `script.js` (optional if separated)
4. Open `index.html` in any modern web browser.

## 🌍 Browser Compatibility
Works best on:
- Google Chrome

For the best experience, use the latest browser version.
