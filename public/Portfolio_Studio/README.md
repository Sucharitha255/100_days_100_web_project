# 🎨 Portfolio Studio

A modern **Portfolio Builder Studio** built with **HTML, CSS, and JavaScript** that enables users to create, customize, preview, and export beautiful developer portfolios in real time.

Portfolio Studio provides a live editing environment where profile details, projects, themes, and personal information can be modified instantly through an intuitive dashboard. The application automatically generates a production-ready portfolio that can be exported as a standalone HTML file.

---

# ✨ Features

* 🎨 Real-time portfolio preview
* ⚡ Live editing without page refresh
* 📂 Dynamic project management
* ➕ Add unlimited projects
* 🗑️ Remove projects instantly

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/100_daus_100_project.git
```

## Navigate

```bash
cd Portfolio-Studio
```

## Run

Simply open **index.html** in your preferred web browser.

No installation or external dependencies are required.
