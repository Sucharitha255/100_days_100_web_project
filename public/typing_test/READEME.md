## 📖 Overview

Typing Test is a responsive web application that helps users improve their typing speed, accuracy, and consistency. The application provides an interactive typing experience with real-time feedback and performance tracking.

---

## ✨ Features

- ⚡ Real-time typing speed (WPM)
- 🎯 Accuracy calculation
- ⏱️ Timer support
- 📊 Live statistics
- 📱 Responsive design
- 🎨 Clean and modern UI
- 🔄 Restart typing test
- 💻 Works on all modern browsers

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
