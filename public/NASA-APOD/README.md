
# 🌌 NASA APOD Explorer (Day‑XX)

## 🚀 Overview
**NASA APOD Explorer** is a lightweight browser extension/web app that fetches and displays the **Astronomy Picture of the Day (APOD)** directly from NASA’s public API.  
It provides stunning daily space imagery along with detailed descriptions, making astronomy accessible and engaging for everyone.

---

## ✨ Features
✅ Fetches NASA’s Astronomy Picture of the Day via API  
✅ Displays high‑resolution images with titles and explanations  
✅ Popup interface for quick daily viewing  
✅ Extension icons for easy access  
✅ Responsive design for different screen sizes  
✅ Beginner‑friendly implementation with clean code  

---

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (ES6)  
- NASA APOD API

## ▶️ How to Run Locally
1. Clone the repository:
```bash
git clone https://github.com/dhairyagothi/100_days_100_web_project.git
```
2. Navigate to project folder:
```bash
cd 100_days_100_web_project/public/NASA-APOD
```
3. Load extension in browser:
   - Open **Chrome/Edge** → Extensions → Enable Developer Mode  
   - Click **Load unpacked** → Select `NASA-APOD` folder  
   - Click the extension icon to view APOD
