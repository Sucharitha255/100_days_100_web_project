# Smart Medicine Reminder & Health Tracker

A beautifully designed, premium web application built to help users manage their daily medications and track their health notes. 

## Features
- 💊 **Add/Edit/Delete Medicines**: Manage your medicine schedules easily.
- 🕒 **Daily Reminders**: Mark medicines as taken; automatically resets the next day.
- 📊 **Progress Dashboard**: View your daily completion percentage in real-time.
- 📝 **Health Notes**: Keep track of symptoms or doctor advice. Auto-saves as you type!
- 🎨 **Premium UI/UX**: Built with a stunning Glassmorphism design and micro-animations.
- 🌓 **Dark/Light Mode**: Smooth toggling between themes with CSS variables.
- 💾 **Local Storage**: All your health data stays completely private and is saved on your device.

## How to Use
Simply open `index.html` in your web browser. 

Click the **Add New** button to add a medicine. Enter the name, dosage, and time. Once added, you can click the checkbox to mark the medicine as taken for the day!

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Grid, Animations, Glassmorphism)
- Vanilla JavaScript (DOM manipulation, LocalStorage API)
- Phosphor Icons
