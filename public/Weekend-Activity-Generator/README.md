# 🎉 Weekend Activity Generator

Weekend Activity Generator is a fun web application that helps users discover exciting activities to make their weekends more enjoyable. Whether you're looking for something relaxing, adventurous, creative, or productive, the app suggests activities instantly.

---

## ✨ Features

- 🎲 Generate random weekend activity suggestions
- 🌍 Variety of activity categories
- ⚡ Instant recommendations
- 📱 Responsive user interface
- 🎨 Clean and modern design
- 🚀 Lightweight and beginner-friendly project

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## 🚀 Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Navigate to the project directory.

3. Open `index.html` in your browser or run it using Live Server.

---

## ▶️ Usage

- Launch the application.
- Click the generate button.
- Get a random weekend activity suggestion.
- Generate as many new ideas as you'd like.
