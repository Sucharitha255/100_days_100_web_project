# 🧘 Overview

**Guided Zen Breathing Visualizer** is an interactive mindfulness application designed to help users relax, improve focus, reduce stress, and regulate breathing through animated visual guidance.

The breathing sphere expands and contracts in rhythm while timers, calming colors, and optional meditation chimes guide every breathing cycle.

Everything runs directly inside the browser without requiring external libraries or a backend.

---

# ✨ Features

## 🌬️ Multiple Breathing Techniques

Choose from three guided breathing methods:

### 🟦 Box Breathing (4–4–4–4)

Perfect for:

* Stress reduction
* Mental clarity
* Improving concentration
* Anxiety management

Cycle:

* Inhale – 4 seconds
* Hold – 4 seconds
* Exhale – 4 seconds
* Hold – 4 seconds

---

### 🌙 Relaxing Pattern (4–7–8)

Designed for:

* Better sleep
* Deep relaxation
* Calming the nervous system

Cycle:

* Inhale – 4 seconds
* Hold – 7 seconds
* Exhale – 8 seconds

---

### ⚖️ Equal Breathing (4–4)

Ideal for:

* Daily meditation
* Focus enhancement
* Balanced breathing

Cycle:

* Inhale – 4 seconds
* Exhale – 4 seconds

---

# 🌈 Animated Breathing Sphere

The application features a beautiful animated breathing sphere that:

* Expands while inhaling
* Holds its size during breath retention
* Contracts while exhaling
* Smoothly transitions between breathing phases
* Changes color according to the current breathing action

---

# ⏱️ Live Countdown Timer

Each breathing step includes:

* Real-time countdown
* Second-by-second updates
* Visual timing feedback
* Automatic progression through breathing cycles

---

# 🔊 Meditation Chimes

Built-in synthesized meditation sounds using the **Web Audio API**.

Different tones are played for:

* 🌬️ Inhale
* ✋ Hold
* 🍃 Exhale

Users can:

* Enable chimes
* Disable chimes anytime
* Continue breathing silently

No external audio files are required.

---

# ▶️ Guided Session Controls

Simple controls include:

* ▶️ Start Session
* ⏸ Pause Session
* 🔊 Toggle Chimes
* Switch breathing pattern anytime

Changing breathing patterns automatically resets the current session for a seamless experience.

# 🎯 How to Use

1. Open the application.
2. Select your preferred breathing pattern.
3. Click **Start Guided Session**.
4. Follow the animated breathing sphere.
5. Breathe according to the displayed instructions.
6. Enable or disable meditation chimes as desired.
7. Pause the session whenever needed.
