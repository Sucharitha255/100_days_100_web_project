# 📱 QR Code Generator

A simple and responsive QR Code Generator that instantly creates QR codes from any text or URL. The application provides a clean interface for generating scannable QR codes that can be used for websites, contact details, notes, and more.

---

## ✨ Features

* Generate QR codes from any text or URL
* Instant QR code generation
* Clean and responsive user interface
* Supports multiple input formats
* Download generated QR code (if supported)
* Lightweight and fast

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* QR Code Generation Library (QRCode.js or equivalent)

---

## 🚀 How to Run Locally

1. Clone the repository.

2. Navigate to the project directory.

3. Open the **QR_Code_Generator** folder.

4. Launch `index.html` in your preferred web browser.

No additional installation or server setup is required.
