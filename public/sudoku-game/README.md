# 🧩 Sudoku Game

Sudoku Game is an interactive web-based puzzle game where players solve a classic 9×9 Sudoku board by filling every row, column, and 3×3 sub-grid with the numbers 1–9 without repetition. The project provides a clean and responsive interface for practicing logical thinking and problem-solving skills.

---

## ✨ Features

- 🧩 Classic 9×9 Sudoku gameplay
- ✍️ Interactive game board
- ✅ Input validation
- 🔄 Restart / Reset game
- 📱 Responsive layout
- 🎨 Clean and modern interface
- ⚡ Lightweight and fast

## ▶️ How to Play

1. Launch the game.
2. Fill the empty cells with numbers from **1–9**.
3. Ensure that:
   - Each row contains unique numbers.
   - Each column contains unique numbers.
   - Each 3×3 box contains unique numbers.
4. Complete the puzzle successfully.

---

## 🎯 Objective

Solve the Sudoku puzzle while following all Sudoku rules and complete the board correctly.
