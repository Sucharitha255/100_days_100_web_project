# ⌨️ Typing Test

A modern and interactive Typing Test application that helps users measure their typing speed and accuracy in real time. The project provides an engaging interface for practicing typing while displaying useful performance metrics such as Words Per Minute (WPM), accuracy, and typing progress.

---

## ✨ Features

* Real-time typing test
* Words Per Minute (WPM) calculation
* Typing accuracy tracking
* Character and mistake counter
* Timer-based typing session
* Responsive user interface
* Instant performance feedback

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## 🚀 How to Run Locally

1. Clone the repository.

2. Navigate to the project folder.

3. Open the **Typing_Test** directory.

4. Open `index.html` in your browser.

No installation or additional dependencies are required.
