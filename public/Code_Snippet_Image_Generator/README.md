# Code Snippet to Image Generator Tool

A beautiful, modern, glassmorphism-based utility tool that allows developers to convert their raw code snippets into stunning, customizable images with macOS-style window frames.

## 🌟 Features
- **Syntax Highlighting**: Real-time highlighting using Prism.js.
- **Multiple Languages**: Support for JavaScript, HTML, CSS, Python, TypeScript, and Java.
- **Themes**: Switch between Dark (Tomorrow), Okaidia, Light, and Twilight themes.
- **Customizable Backgrounds**: 5 stunning gradient presets and a custom color picker.
- **High-Quality Export**: Export your snippets instantly as high-resolution PNG images.
