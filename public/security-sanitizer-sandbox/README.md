# 🔐 Security Sanitizer Sandbox

Security Sanitizer Sandbox is a browser-based web application that demonstrates how user input can be sanitized to prevent common web security vulnerabilities such as Cross-Site Scripting (XSS). It provides an interactive environment to understand secure input handling and safe rendering techniques.

---

## ✨ Features

- 🔒 Input sanitization demonstration
- 🛡️ XSS prevention examples
- ⚡ Real-time sanitization preview
- 🧪 Interactive sandbox environment
- 📱 Responsive design
- 🎨 Clean and intuitive user interface

---

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd security-sanitizer-sandbox
```

Run locally

- Open `index.html`
- OR use Live Server in Visual Studio Code

## ▶️ Usage

1. Launch the application.
2. Enter text or HTML input.
3. Observe the sanitized output.
4. Compare raw and sanitized content.
5. Learn how proper sanitization protects web applications.
