# CSS Glassmorphism & Neumorphism UI Generator

A premium, interactive developer tool to instantly generate modern CSS UI effects.

## 🌟 Features
- **Dual Modes:** Seamlessly switch between Glassmorphism (Frosted Glass) and Neumorphism (Soft UI) generators.
- **Real-Time Preview:** Watch the effect change instantly as you adjust the sliders.
- **Customizable Properties:** 
  - *Glassmorphism:* Adjust Blur, Transparency, Outline thickness, and Tint Color.
  - *Neumorphism:* Adjust Shadow Distance, Blur Radius, Shadow Intensity, and Background Color.
- **One-Click Copy:** Click the copy button to instantly grab the cross-browser compatible CSS code to use in your own projects.
- **Premium Design:** Built with a beautiful dark theme, using Phosphor Icons and modern typography (Inter & JetBrains Mono).
