# Email Validator

## 📌 Description
Email Validator is a simple yet powerful frontend project that ensures users enter a properly formatted, real, and usable email address before submission.  
It helps prevent invalid or fake emails in forms, improving data quality and user experience.

## ✨ Features
- Validates email format using regex and HTML5 `input type="email"`
- Real-time validation with instant feedback
- Error messages for invalid or empty inputs
- Clean and responsive UI with header, navbar, and footer
- Lightweight implementation with pure HTML, CSS, and JavaScript

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript ES6

## 🚀 Installation / Setup
1. Clone or download this project.
2. Open the project folder.
3. Ensure the following files are present:
   - `index.html`
   - `style.css`
   - `script.js`
4. Open `index.html` in any modern web browser.

## 🌍 Browser Compatibility
This project works best on:
- Google Chrome
- Mozilla Firefox
- Safari
- Microsoft Edge

For the best experience, use the latest browser version.

## 🎯 Usage
1. Enter an email address in the input field.
2. The validator checks format instantly.
3. If invalid, an error message is displayed.
4. If valid, the form can be submitted successfully.
