## 📄 About The Project

**DocNow** is a highly interactive, client-side card management dashboard built entirely with vanilla web technologies. It helps users declutter their digital lives by allowing them to store and categorize notes, important URLs, and reminders. The app features a beautiful UI with an animated mesh background and ensures your data is never lost by persisting it locally in your browser. 

---

## ✨ Key Features

*   **Smart Categorization:** Organize your cards into color-coded categories like Note, Link, Reminder, Reference, and Work.
*   **Advanced Search & Filter:** Instantly find what you need using the search bar or by toggling category filters.
*   **Theme Toggle:** Built-in Light and Dark mode support that remembers your preference.
*   **Flexible Views:** Switch seamlessly between Grid and List view layouts for your dashboard.
*   **Data Persistence:** All cards are automatically saved to your browser's `localStorage`, ensuring they remain when you return.
*   **Keyboard Shortcuts:** Boost productivity with shortcuts like `Ctrl/Cmd + K` to search and `Esc` to close modals.
*   **Live Statistics:** Track your card count in real-time, categorized by Total, Showing, and Today.

## 🛠️ Technologies Used

*   **HTML5:** Semantic markup for the dashboard layout, navigation, and modal dialogues.
*   **CSS3:** Custom styling utilizing CSS variables for theme switching, flexbox/grid for layouts, and backdrop-filters for a modern glassmorphism effect.
*   **JavaScript (Vanilla):** Dynamic DOM manipulation for rendering cards, managing state, handling form submissions, and interacting with the Web Storage API.

---

## 🚀 How to Run Locally

1. **Clone the repository** to your local machine.
2. **Navigate** to the `public/DocNow` directory.
3. **Open the `index.html` file** directly in any modern web browser.
   * *Note: No external dependencies, frameworks, or build tools are required to run this project!*
