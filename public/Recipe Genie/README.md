# 🍳 Recipe Genie

### Discover • Search • Explore • Cook

## 📌 Overview

Recipe Genie is a responsive web application that allows users to discover recipes, search meals, browse categories, and access detailed cooking instructions through a clean and interactive interface.

---

## ✨ Key Features

### 🔍 Smart Recipe Search
- Search meals instantly.
- Quick and responsive results.

### 🍽 Category Exploration
- Browse recipes by categories.
- Easy navigation between sections.

### 📖 Detailed Recipe Page
- Meal information.
- Ingredients list.
- Cooking instructions.
- Recipe details.

### 📱 Responsive Design
- Mobile-friendly UI.
- Tablet support.
- Desktop optimized.

### ⚡ Modern User Experience
- Fast loading pages.
- Clean interface.
- Smooth navigation.

Clone the repository:


Navigate to the project:

```bash
cd public/Recipe\ Genie
```

Run the project:

```bash

```
