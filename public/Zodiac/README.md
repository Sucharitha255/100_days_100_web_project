# 🔮 Zodiac

A modern Zodiac web application that allows users to explore zodiac signs, their characteristics, personality traits, strengths, weaknesses, compatibility, and related constellations through an interactive interface.

---

## Features

- View all 12 zodiac signs
- Detailed zodiac information
- Interactive user interface
- Responsive design
- Constellation information
- Clean and modern layout

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

---

## Installation

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Run `index.html` using Live Server or any local server.

---

## Usage

- Open the application.
- Select or explore zodiac signs.
- View personality traits and constellation details.
