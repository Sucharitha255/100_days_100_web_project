# 🐍 Snake Game

A modern and responsive Snake Game built using HTML5, CSS3, and JavaScript. Control the snake, collect food, increase your score, and avoid collisions with walls or yourself. The project demonstrates game development fundamentals using HTML Canvas and JavaScript.

---

# 🎮 Features

- Classic Snake gameplay
- Smooth keyboard controls
- Score tracking
- Increasing challenge as the game progresses
- Collision detection
- Game Over screen
- Restart functionality
- Responsive design
- Lightweight and fast performance
- Clean and modern UI

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- HTML Canvas API

# 🎮 Controls

| Key | Action |
|------|---------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

---

# 🏆 Game Rules

- Eat food to grow the snake.
- Every food increases your score.
- Avoid hitting the walls.
- Avoid colliding with your own body.
- The game ends when a collision occurs.
