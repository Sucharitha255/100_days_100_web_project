# 📅 Digital Planner

## About the Project

Digital Planner is a browser-based productivity application that helps users organize their daily tasks and activities in an interactive way.

The planner includes visual customization using stickers and audio support to create an engaging planning experience. Everything runs directly in the browser without requiring any installation.

## Features

- 📝 Create and manage planner entries
- 🎵 Audio support for interactive feedback
- 🌟 Decorative sticker functionality
- 📱 Responsive design
- ⚡ Fast and lightweight
- 🎨 Clean user interface

## 🌌 About The Project

The **Digital Planner** is a comprehensive productivity dashboard built entirely with vanilla web technologies. It goes beyond a simple to-do list by offering a personalized planning experience. Users can organize their schedules using an interactive calendar, stay focused with a built-in ambient audio player, and customize their planning space using a dynamic sticker system.

---

## ✨ Key Features

* 🗓️ **Interactive Calendar:** A robust calendar system to track days, events, and schedules without visual overlap or clutter.
* 🎵 **Ambient Audio Player:** Built-in audio controls to play focus-enhancing background sounds while you plan and work.
* 🏷️ **Customizable Stickers:** A fun, interactive sticker system that allows users to drag, drop, and place decorative or status-based stickers onto their planner.
* ✅ **Task Management:** Core planning features for adding, tracking, and completing daily goals and to-dos.
* 🎨 **Modern UI/UX:** A beautifully styled interface with smooth transitions, responsive design, and intuitive navigation.

## 📁 Project Structure

```text
Digital-Planner/
├── assets/         # Directory containing images, audio files, and sticker graphics
├── index.html      # Main semantic structure and layout of the planner
├── style.css       # Core styling, responsive layouts, and UI animations
├── app.js          # Main application logic, state management, and calendar rendering
├── audio.js        # Logic for the ambient background audio player
├── stickers.js     # Functionality for selecting, dragging, and dropping stickers
└── README.md       # Project documentation
```
## 🛠️ Technologies Used

* **HTML5:** Semantic markup for the planner's layout and integrated audio elements.
* **CSS3:** Custom styling, flexbox/grid layouts for the calendar, and interactive hover states.
* **JavaScript (ES6+):** DOM manipulation for dynamic calendar generation, event listeners for the drag-and-drop sticker mechanics, and HTML5 Audio API integration for the ambient sound player.
