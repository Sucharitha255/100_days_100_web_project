# 🌦️ Weather App

A modern weather application that provides real-time weather information for any city using the OpenWeatherMap API. The application features a clean interface with current weather conditions, temperature, humidity, wind speed, and forecast details.

---

## 🚀 Features

* Search weather by city name
* Real-time weather information
* Temperature display
* Humidity information
* Wind speed details
* Weather condition icons
* Responsive design
* Fast API-based weather updates

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* OpenWeatherMap API

---

## 🌐 API Used

**OpenWeatherMap API**

* Current Weather API
* Documentation: https://openweathermap.org/api

> **Note:** An API key is required to use this application.

---

## ▶️ Run Locally

1. Clone the repository.

2. Navigate to the project folder.

3. Obtain a free API key from OpenWeatherMap.

4. Add your API key inside the JavaScript configuration.

5. Open `index.html` in your browser.
