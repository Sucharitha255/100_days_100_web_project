# ⭐ ReviewForm

ReviewForm is a simple and responsive web application that allows users to submit reviews and ratings through an intuitive interface. It demonstrates form handling, client-side validation, and interactive UI design using modern web technologies.

---

## ✨ Features

- ⭐ Star rating system
- 📝 User review submission
- ✅ Client-side form validation
- 📱 Responsive design
- 🎨 Clean and modern interface
- ⚡ Instant feedback on form submission

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd ReviewForm
```

Run locally

- Open `index.html` in your browser
- OR use Live Server in Visual Studio Code

## ▶️ Usage

1. Launch the application.
2. Fill in the review form.
3. Provide a rating.
4. Submit your feedback.
5. View confirmation after successful submission.
