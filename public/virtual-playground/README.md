# 🌍 Virtual Playground

> **An interactive physics playground where you can experiment with gravity on different planets, customize the ball's appearance, and watch realistic bouncing physics in action.**

## 🎮 Overview

**Virtual Playground** is a fun browser-based physics simulation that demonstrates how gravity changes across different planets. Click the ball to launch it into the air, switch between planetary gravities, and customize its appearance with different colors and shapes.

This project is perfect for learning basic physics concepts, JavaScript animations, and DOM manipulation while having fun.

---

## ✨ Features

### 🌍 Planet Gravity Simulation

Experience different gravitational forces by switching between planets:

* 🌕 Moon
* 🔴 Mars
* 🌍 Earth
* 🪐 Saturn
* 🟠 Jupiter

Each planet changes the ball's movement instantly without restarting the simulation.

---

### ⚽ Interactive Physics Engine

The playground includes a realistic physics simulation featuring:

* Gravity acceleration
* Elastic bouncing
* Collision detection
* Wall boundaries
* Energy loss after each bounce
* Random horizontal movement

---

### 🎯 Ball Interaction

Simply click the ball to:

* Launch it upward
* Apply random horizontal velocity
* Watch realistic bouncing based on the selected planet

---

### 🎨 Ball Customization

Personalize the ball with:

#### 🎨 Color Picker

Choose any color using the built-in color picker.

#### 🔷 Shape Selector

Available shapes include:

* ⚪ Circle
* ◼️ Square
* 🔺 Triangle
* ⭐ Star

Shapes are created using modern CSS `clip-path` techniques.

---

### 🔄 Reset Function

The Reset button instantly restores:

* Ball position
* Ball velocity

Your selected:

* Planet
* Ball color
* Ball shape

remain unchanged.

---

## 🚀 How It Works

1. Select your favorite planet.
2. Click the ball to launch it.
3. Observe how gravity affects the movement.
4. Change planets anytime.
5. Customize the ball's color and shape.
6. Reset whenever you want to start over.
