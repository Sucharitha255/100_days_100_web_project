# Color Jump Game

A fast-paced, minimalist jumping game with a modern CuboSapiens-inspired UI, responsive controls, and high-score tracking.

## Features
- **Responsive Design:** Playable on Desktop, Tablet, and Mobile.
- **Touch & Keyboard Controls:** Tap the screen, click the mouse, or press the `Spacebar` to jump.
- **Progressive Difficulty:** Obstacles speed up as your score increases.
- **Persistent Storage:** High scores are saved using `localStorage`.
- **Modern UI:** Built with sleek CSS variables, dynamic shadows, and glowing elements.

## Tech Stack
- HTML5 Canvas
- CSS3 (Variables, Flexbox)
- Vanilla JavaScript

## How to Play
1. Click **Start Game**.
2. Tap the screen or press the Spacebar to keep the glowing square in the air.
3. Dodge the colored blocks coming from the right.
4. Try to beat your high score!
