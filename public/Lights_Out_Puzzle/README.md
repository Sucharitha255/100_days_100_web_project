
# 💡 Lights Out Puzzle (Day‑84)

## 🚀 Overview
Lights Out Puzzle is a classic logic‑based game built with **HTML, CSS, and JavaScript**. The objective is simple yet challenging: toggle lights on a grid until all are turned off. Each move flips the selected light and its adjacent neighbors, creating a fun and brain‑teasing experience.

---

## ✨ Features
✅ Interactive grid‑based puzzle  
✅ Click‑to‑toggle lights with neighbor flipping logic  
✅ Win detection when all lights are off  
✅ Responsive design for desktop and mobile  
✅ Smooth animations and hover effects  
✅ Beginner‑friendly implementation with clean code  

---

## 🎮 Gameplay Overview
- **Grid Layout**: Lights arranged in a square grid.  
- **Toggle Mechanism**: Clicking a light flips it and its adjacent lights.  
- **Objective**: Turn off all lights to win.  
- **Challenge**: Each move affects multiple lights, requiring strategy and foresight.
